✨🎩 What is Selectors ?✨🎩

✨Selectors means:- Suppose you want to select a specific element or specific part of the webpage is called a selectors🪄.

✨🎩 Types Of Selectors ✨🎩

🚀Types of selectors are follows as under:-

(i)✨Element Selectors
(ii)🌟Class Selectors
(iii)🌈 ID Selectors
(iv)🎨Descendant Selectors
(v)🖌️Child Selectors
(vi)🧷attribute selectors

✨ What is element selectors ✨?

🚀 Element Selectors Means :- suppose you want to select a specific tag like h1 or p at that time we use a Element Selectors
Example:- p{ color :blue; }

🌟 What is Class Selectors 🌟?

🚀 Class Selectors Means :- if we want to select element one or more than time in one webpage is called a class selector for select class selector we use a symbol that is dot symbol
Example:- .new{ color :blue; }

🌟 What is ID Selectors 🌟?

🚀 ID Selectors Means :- ID selectors helps to select a element only one time in webpage because it is unique element and we can use only one time in a webpage. For this we can use a # symbol
Example:- #heading{ color :blue; }

🌟 What is attribute selectors 🌟?

🚀 attribute selectors Means :- Selects all input elements with the "required" attribute *
Example :-
input[required] { border: 1px solid red; }
